# Gitlet
UC Berkeley CS61B - "Data Structures and Algorithms" Project

We implemented a lite version of the version control system, GIT, mimicking main functions including: init, add, commit, rm, log, global-log, find, status, checkout, branch, rm-branch, reset, and merge.

Language - Java
Libaries/Packages- java.io : File, IOException, charset.StandardCharsets, BufferedOutputStream, ByteArrayOutputStream, FileInputStream, FilenameFilter, IOException, ObjectInputStream, ObjectOutputStream, Serializable, file.Files, file.Paths, charset.StandardCharsets, MessageDigest, SuchAlgorithmException, Arrays, Formatter;
(Used for reading and sending system inputs and outputs through data streams, serialization and the file system)
